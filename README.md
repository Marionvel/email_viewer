## En local
1. Se positionner sur le dossier avec le terminal
2. Lancer $ bundle install
3. Lancer $ rails db:migrate
4. Lancer $ rails db:seed
5. Lancer $ rails server
6. Se rendre sur localhost:3000 avec un navigateur
